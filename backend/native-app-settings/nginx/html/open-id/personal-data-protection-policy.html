個人情報保護方針

株式会社サンプルホールディングス（以下「当会社」という）は、個人情報の保護が重要であることを理解し、個人情報を正しく扱うことが当会社の重要な責務であると認識し、以下の方針に従い、個人情報の保護に努めお客さまの信頼に応えてまいります。

  1. 当会社は、個人情報の保護に関する法令およびその他の規範を遵守し、利用及び提供に関して規程及び具体的ルールを定め、実施します。
  2. 当会社が個人情報を取得する際には、利用目的を明確化し、収集した個人情報の使用範囲を限定して適切に取り扱います。
  3. 当会社は、個人情報の紛失、改ざん、漏えいなどを防止するための適切な措置を講じます。
  4. 当会社は、本人の情報について、開示、訂正又は削除などを求められたときは、法令に従って速やかに対応します。
  5. 当会社は、個人情報を保護するためのコンプライアンス・プログラムを策定し、これを維持し、継続的に改善ます。
